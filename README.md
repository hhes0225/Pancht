# Pancht
# 프로젝트 소개
- 일대일 멀티 Yacht 게임의 온라인 서버를 제작한 프로젝트입니다.
- 3개의 API 서버(APi Account 서버, Api Game 서버, Matching 서버)와 1개의 소켓 서버(Yacht 로직 서버)로 구성되어 있습니다.

---

# 프로젝트 개요
- 개발 기간: 2024.08 -
- 참여 인원: 4인(기획 1명, 아트 1명, 서버 1명, 클라이언트 1명)
- 참여 역할: 서버
- 사용 언어: C#
- 사용 도구: ASP.NET Core 8.0, MySQL, Redis
